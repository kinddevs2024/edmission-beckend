# Ассистент на DeepSeek — без Ollama

Ассистент может работать **без Ollama**: достаточно ключа API DeepSeek. Ничего ставить и скачивать не нужно.

## Шаг 1. Получить API-ключ

1. Зайди на **https://platform.deepseek.com**
2. Зарегистрируйся или войди
3. Раздел **API Keys** → **Create API Key**
4. Скопируй ключ (начинается с `sk-...`)

## Шаг 2. Подключить на проекте

В файле **`.env`** бэкенда добавь или измени:

```env
DEEPSEEK_API_KEY=sk-твой_ключ_сюда
```

Сохрани файл и перезапусти бэкенд.

## Готово

- Ассистент в интерфейсе (кнопка чата / страница Edmission AI) будет отвечать через DeepSeek.
- Ollama не нужен.
- Если `DEEPSEEK_API_KEY` не задан, бэкенд будет пытаться использовать Ollama (как раньше).

## Локальный запуск

1. Запусти MongoDB (если нужна локальная база).
2. В `.env` задай `DEEPSEEK_API_KEY=sk-...`.
3. В папке бэкенда: `npm run dev` (или `node dist/index.js`).
4. Фронт: `npm run dev` в папке фронта.
5. Открой приложение, войди и нажми на кнопку чата или перейди в раздел Edmission AI — отправь сообщение и проверь ответ.

На сервере то же самое: в переменных окружения задай `DEEPSEEK_API_KEY`, перезапусти бэкенд — ассистент будет работать без Ollama.
